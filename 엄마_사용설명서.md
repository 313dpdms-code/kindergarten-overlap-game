# 🎨 겹친 그림 맞추기 게임 - 사용 설명서

엄마, 이 게임은 어린이집 수업에서 아이들과 함께 즐기실 수 있는 "겹친 그림 맞추기" 게임이에요.

---

## 🎮 어떤 게임인가요?

화면 가운데에 **여러 종류의 그림**이 살짝 겹쳐서 반투명하게 나타나요.
아이들이 "여기 뭐가 숨어 있을까~?" 하고 맞히면, 엄마가 [정답 공개] 버튼을 눌러
**모든 그림을 한 번에** 펼쳐서 정답을 보여주는 방식이에요.

- **한 라운드** = 그림 3장 또는 5장 맞히기 (시작할 때 선택)
- **한 게임** = 5라운드 진행 후 축하 화면

### 🎚️ 시작 화면에서 고를 수 있는 옵션

| 옵션 | 선택지 | 설명 |
|------|--------|------|
| **그림 개수** | 3장 / 4장 / 5장 | 한 라운드에 몇 장을 맞힐지. 3장은 쉬움, 4장은 보통, 5장은 도전 |
| **색상 모드** | 🎨 컬러 / ⚫ 흑백 | 컬러는 색깔 힌트 있어서 쉬움. 흑백은 검은 실루엣만 보여서 더 어려움 |

---

## 💻 게임 실행하는 방법

### 처음 한 번만 - 컴퓨터 준비

이 게임은 컴퓨터에서 작은 서버를 켜서 인터넷 주소(localhost)로 접속하는 방식이에요.
어렵게 들리지만 실은 **명령어 두 줄**이면 됩니다.

#### 1단계: 터미널 열기
- 맥북에서 `command(⌘) + space` 누르고 "터미널" 검색 → 엔터
- 또는 응용프로그램 → 유틸리티 → 터미널

#### 2단계: 게임 폴더로 이동
터미널에 아래 명령어를 그대로 복사해서 붙여넣고 엔터:
```
cd ~/Desktop/overlap-game
```

#### 3단계: 게임 켜기
```
npm run dev
```

잠깐 기다리면 아래와 같은 줄이 나와요:
```
✓ Ready in 2.3s
- Local: http://localhost:3000
```

#### 4단계: 브라우저(크롬/사파리) 열고 접속
주소창에 입력:
```
http://localhost:3000
```

✨ **끝! 게임 화면이 뜨면 성공이에요.**

---

### 게임 끄는 법

수업이 다 끝나면 터미널 창을 클릭해서 활성화한 다음:
- `Ctrl + C` 키를 누르면 서버가 꺼져요.
- 터미널 창을 그냥 닫아도 자동으로 꺼져요.

## 🎯 게임 화면 사용법

### 시작 화면
- 큰 분홍색 글씨 **"겹친 그림 맞추기"** 가 보이고
- 가운데에 **그림 개수 (3장/5장)** 와 **색상 모드 (컬러/흑백)** 토글이 있어요. 원하는 옵션을 누르세요.
- 아래에 **🚀 시작하기** 초록 버튼을 누르면 게임이 시작돼요.

### 게임 화면 (1라운드 = 그림 3장)

화면에 나오는 버튼은 딱 **3개**예요:

| 버튼 | 위치 | 언제 누르나요? |
|------|------|----------------|
| **🏠** | 우측 상단 | 도중에 처음 화면으로 돌아갈 때 (확인 메시지가 뜸) |
| **✅ 정답 공개** | 하단 왼쪽 (파란색) | 아이들이 답을 말한 뒤 정답을 보여줄 때 |
| **➡ 다음 문제** | 하단 오른쪽 (초록색) | 그림 3장 모두 공개된 후 다음 라운드로 넘어갈 때 |

### 게임 흐름

1. 화면에 그림 3장(또는 5장)이 겹쳐서 반투명하게 나타나요
2. 아이들에게 **"이 안에 뭐가 숨어 있을까?"** 물어봐요
3. 아이들이 자유롭게 외쳐요: "강아지!" "사과!" "자동차!"
4. 충분히 답을 들으셨으면 **[✅ 정답 공개]** 버튼을 한 번 누르세요
   - **모든 그림이 한 번에** 가로로 나란히 펼쳐지며, 그림 아래에 이름이 표시돼요
5. 아이들과 답을 확인하셨으면 **[➡ 다음 문제]** 버튼(통통 튀고 있음)을 누르세요
6. 5라운드를 다 마치면 🎉 축하 화면이 떠요

## 🎲 어떤 그림들이 나오나요?

매 라운드마다 **다섯 카테고리에서 그림을 섞어** 보여드려요.
- **3장 모드**: 다섯 카테고리 중 무작위로 3개를 골라 1장씩 (가장 쉬움)
- **4장 모드**: 다섯 카테고리 중 무작위로 4개를 골라 1장씩 (적당한 난이도)
- **5장 모드**: 다섯 카테고리에서 모두 1장씩 (항상 골고루 등장, 가장 어려움)

| 카테고리 | 들어있는 그림들 |
|---------|----------------|
| 🐶 **동물** (12종) | 강아지, 고양이, 토끼, 곰, 사자, 호랑이, 코끼리, 원숭이, 판다, 돼지, 개구리, 닭 |
| 🍎 **과일** (10종) | 사과, 바나나, 포도, 딸기, 수박, 오렌지, 복숭아, 파인애플, 레몬, 체리 |
| 🚗 **탈것** (8종) | 자동차, 버스, 기차, 비행기, 배, 자전거, 트럭, 헬리콥터 |
| 🍔 **음식** (8종) | 케이크, 피자, 빵, 햄버거, 아이스크림, 사탕, 도넛, 쿠키 |
| 🌸 **자연** (8종) | 해, 달, 별, 구름, 무지개, 꽃, 나무, 눈송이 |

그림은 **트위터의 무료 공개 일러스트(Twemoji)** 를 사용해서, 모든 컴퓨터에서 똑같이 예쁘게 보여요.

## 💡 수업에서 활용 팁

- **TV/빔 프로젝터** 에 화면을 띄우고 노트북으로 조작하시면 아이들 모두가 잘 볼 수 있어요
- 아이들이 어려워하면 그림을 가리키며 **"이 빨간 동그란 건 뭐일까?"** 같은 힌트를 직접 주세요
- 정답 공개 후 그림 이름을 큰 글씨로 띄워주니 **단어 학습**도 자연스럽게 돼요
- 너무 빨리 공개하지 말고, 아이들이 충분히 추측할 시간을 주세요

### 🎚️ 난이도 조절 팁
| 상황 | 추천 옵션 |
|------|-----------|
| 만 3~4세 아이들, 처음 시작 | **3장 + 🎨 컬러** (가장 쉬움) |
| 만 4~5세 아이들 | **4장 + 🎨 컬러** (적당함) |
| 만 5~6세 아이들 | **5장 + 🎨 컬러** |
| 익숙해진 아이들 / 도전 모드 | **3장 또는 4장 + ⚫ 흑백** (실루엣만 보고 맞히기) |
| 만 6~7세, 어려운 도전 | **5장 + ⚫ 흑백** (가장 어려움) |

## 🐛 문제가 생기면

| 증상 | 해결 방법 |
|------|-----------|
| `npm run dev` 가 "command not found" 라고 나와요 | 컴퓨터에 Node.js가 안 깔려 있어요. https://nodejs.org 에서 LTS 버전 설치해주세요 |
| 그림이 안 떠요 (네모만 보여요) | 인터넷 연결 확인해주세요. 그림은 인터넷에서 불러오거든요 |
| 화면이 너무 작아요 | 브라우저에서 `Cmd(⌘) + +` 키 눌러서 확대하시거나, F11(전체 화면) 눌러보세요 |
| 다른 문제가 생기면 | 예은이한테 카톡 주세요 😊 |
